1. Print Spiral Matrix
Problem:
Given a 2D matrix, print it in spiral order.
Input:
1  2  3  
4  5  6  
7  8  9
Output:
1 2 3 6 9 8 7 4 5



2. Count Frequency of Characters
Problem:
Given a string, count the frequency of each character and print it in alphabetical order.
Input:
Input: "computercomputer"  
Output: e4 f1 g2 k2 o1 r1 s2

3. Reverse Words in a Sentence
Problem:
Reverse the words in a sentence without reversing individual characters.
Input:
Input: "I write coding"  
Output: "coding write I"


4. Problem:
Write a function to check if two strings are anagrams of each other.
Input:
str1 = "listen", str2 = "silent"  
Output: true


5. Leaders in an Array
Problem:
An element is a leader if it is greater than all elements to its right. Find all such elements.
Input:
Input: {16, 17, 4, 3, 5, 2}  
Output: 17 5 2


6. Merge Two Sorted Arrays
Problem:
Merge two sorted arrays into one sorted array (no duplicates).
Input:
arr1 = {1, 3, 5}, arr2 = {2, 3, 6}  
Output: {1, 2, 3, 5, 6}
